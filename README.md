# Clear My Way

A new Flutter project with firebase.

## How to setup in local system.

```
flutter pub get

flutter run
```

## IF ERROR (FIREBASE)

Use these commands only in the case of errors related to firebase

```
npm install -g firebase-tools

firebase login

flutter pub global activate flutterfire_cli
```
